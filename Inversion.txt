=== Code ===

Dim Mat A
List Ans[1->D 'Nombre d inconnues
Mat A->Mat B 'Recopie de Mat A dans Mat B
Mat B# 'Affichage de Mat B
"DESCENTE"#
For 1->K To D-1
Mat B[K,K->P 'P prend la valeur pivot
If P 'Si P non nul
Then For K+1->I To D
Mat B[I,K->V 'V prend la valeur a annuler
For K->J To 2*D
Mat B[I,J]-V/P*Mat B[K,J->Mat B[I,J 'On remplace la valeur en position I,J
Next
Next
Else "PIVOT NUL"
Stop 'Arrêt du programme
IfEnd
Mat B# 'Affichage de la matrice 
Next
"REMONTEE"#
For D->I To 2 Step -1 'I se decremente de 1 a chaque tour
Mat B[I,I->P 'P prend la valeur pivot
If P 'Si P non nul
Then For 1->K To I-1
Mat B[K,I->V
For I->J To 2*D
Mat B[K,J]-V/P*Mat B[I,J->Mat B[K,J
Next
Next
Else "PIVOT NUL"
Stop 'Arrêt du programme
IfEnd
Mat B# 'Affichage de la matrice
Next
For 1->I To D 'Calcul des solutions
For D+1->J To 2*D
Mat B[I,J]/Mat B[I,I->Mat B[I,J
Next
1->Mat B[I, I
Next
"RES"#
Mat B# 'Affichage des solutions

=== Pour ecrire le code ===

* Menu PRGM puis F3 puis choisir un nom puis EXE

- Le symbole # represente le symbole triangle accessible via SHIFT VARS F5. Il permet d'attendre que l'utilisateur appuie sur EXE pour continuer
- Le symbole ' permet l'écriture de commentaires (non lus par l'interpreteur) et n'est donc pas necessaire a l'execution du code. Il est accessible via F6 puis sixieme caractere sur la droite
- Mat est accessible via OPTN F2 F1
- Dim est accessible via OPTN F2 F6 F2 
- List est accessible via OPTN F1
- If est accessible via SHIFT VARS F1
- For est accessible via SHIFT VARS F6 F1
- Stop est accessible vai SHIFT VARS F2 F4
- Le symbole / est le symbole diviser
- Mat B[I,J equivaut a Mat B[I,J]

=== A propos ===

Calcule la matrice inverse et affiche les differentes etapes.
Ne change pas l'ordre des equations et arrête le programme si le pivot est nul. 

=== Utilisation ===

Le code utilise la matrice A qui doit etre editee dans le menu RUN MAT. Appuyer sur F3 puis entrer les dimensions (m=nombre d'inconnues et n=nombre d'inconnues*2) puis EXE. EXIT pour finir.

Exemple mat A :
3 -7 -2 1 0 0
-3 5 1 0 1 0
6 -4 0 0 0 1

Ensuite aller dans le menu PRGM. Se placer sur le programme et appuyer sur F1.
S'affiche tout d'abord la matrice qui va etre travaillee. Appuyer sur EXE pour passer a la suite a chaque fois. S'affichent ensuite les differentes etapes  pour la triangularisation. La derniere matrice qui s'affiche est le resultat. Celle-ci est visible dans la matrice B depuis RUN MAT. 
 
ATTENTION : Le programme utilise la matrice B. Son precedent contenu sera donc efface.
